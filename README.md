# Hospital_Emergency_Room_Dashboard
Hospital Emergency Room Dashboard is an interactive healthcare analytics dashboard built in Excel/Power BI to monitor patient flow, wait time, admission status, and department referrals. It provides visual insights through KPIs and charts to improve hospital emergency room performance and decision-making.

## Project Objective
The objective of this project is to analyze hospital emergency room performance by tracking patient visits, wait times, admission status, satisfaction scores, and department referrals. The dashboard helps healthcare management improve patient service efficiency and monitor monthly hospital operations through interactive visual reports.

## Questions
1. What is the total number of patients visited each month?
2. What is the average patient wait time in the emergency room?
3. How many patients were admitted vs not admitted?
4. What is the gender-wise distribution of patients?
5. Which age group has the highest number of patients?
